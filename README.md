# DanceUIRuntime

DanceUIRuntime provides the runtime support layer for the DanceUI ecosystem.

- Contains runtime metadata, layout, tuple, and value handling utilities
- Includes tests and an example project for local development
- Ships with CocoaPods configuration and initialization scripts

## Repository Structure

- `DanceUIRuntime/`: Core sources, headers, and tests
- `Example/`: Example project with CocoaPods and Bundler configuration
- `Podspecs/`: Local podspec dependencies
- `init.sh`: Project initialization script
- `DanceUIRuntime.podspec`: CocoaPods specification file
